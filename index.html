Tech Spotlight: Java & Spring Boot

My expertise in Java includes building scalable microservices with Spring Boot and event-driven systems with Apache Kafka. Below is a sample REST API endpoint for a financial transaction service: 

                
package com.timothymoswa.financeservice;

import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.kafka.core.KafkaTemplate;
import org.springframework.web.bind.annotation.*;

@RestController
@RequestMapping("/api/transactions")
public class TransactionController {

    @Autowired
    private TransactionService transactionService;
    @Autowired
    private KafkaTemplate kafkaTemplate;

    @PostMapping
    public Transaction createTransaction(@RequestBody Transaction transaction) {
        Transaction savedTransaction = transactionService.save(transaction);
        kafkaTemplate.send("transactions-topic", savedTransaction);
        return savedTransaction;
    }

    @GetMapping("/{id}")
    public Transaction getTransaction(@PathVariable Long id) {
        return transactionService.findById(id)
                .orElseThrow(() -> new ResourceNotFoundException("Transaction not found"));
    }
}

This endpoint uses Spring Boot for RESTful services and Kafka for event streaming, showcasing dependency injection and error handling.
