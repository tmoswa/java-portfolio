Tech Spotlight: Java & Spring Boot

I build high-throughput microservices using Spring Boot and Apache Kafka for event-driven architectures, integrating with payment systems like ISO 8583. Below is a sample REST API for a payment gateway: 

                
package com.timothymoswa.paymentgateway;

import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.kafka.core.KafkaTemplate;
import org.springframework.web.bind.annotation.*;

@RestController
@RequestMapping("/api/payments")
public class PaymentController {

    @Autowired
    private PaymentService paymentService;
    @Autowired
    private KafkaTemplate kafkaTemplate;

    @PostMapping("/process")
    public Payment processPayment(@RequestBody PaymentRequest request) {
        Payment payment = paymentService.process(request);
        kafkaTemplate.send("payment-events", payment);
        return payment;
    }

    @GetMapping("/{id}")
    public Payment getPayment(@PathVariable String id) {
        return paymentService.findById(id)
                .orElseThrow(() -> new ResourceNotFoundException("Payment not found"));
    }
}

This endpoint demonstrates Spring Boot for RESTful services, Kafka for event streaming, and secure payment processing, aligned with my expertise in FinTech integrations.
